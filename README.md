# ⏱️ DáTempo?

Um app simples que te ajuda a descobrir se dá tempo de estudar tudo antes da prova com base no seu nível, dificuldade da matéria e tempo disponível.

---

## 📌 Sobre o projeto

O **DáTempo?** foi criado para resolver um problema comum:
👉 *“Será que eu consigo estudar tudo antes da prova?”*

Com base em:

* Seu nível atual no assunto
* Dificuldade da matéria
* Quantidade de assuntos
* Horas disponíveis por dia
* Prazo até a prova

O sistema calcula automaticamente se o seu plano é:

* ❌ Inviável
* ⚠️ Possível
* ✅ Super viável

---

## ⚙️ Como funciona

Cada assunto recebe um "peso" baseado em:

* Dificuldade (1 a 5)
* Seu nível (iniciante → avançado)

O sistema calcula o total de horas necessárias e compara com o tempo disponível.

## 🚀 Como rodar o projeto

```bash
git clone https://github.com/LucasMed1/DaTempo
cd DaTempo
```

Depois é só abrir o arquivo `index.html` no navegador.

---

## 🧠 Exemplo de uso

1. Adicione os assuntos que precisa estudar
2. Defina seu nível em cada um
3. Informe quantas horas por dia você tem
4. Coloque o prazo até a prova
5. Clique em **"Ver meu veredito"**
